Description

Random Forests

Parameters (3):

Number of trees

Data type: Integer Domain: 1, Typical value: 10

Meaning: Number of trees

Number of variables per tree

Data type: Integer Domain: (oo, oo) Typical value: 0

Meaning: Number of variables per tree (zero defaults to the square root of the number of layers)

Force unsupervised learning

Data type: Integer Domain: 0, Typical value: 0

Meaning: When absence points are provided, this parameter can be used to ignore them forcing unsupervised learning. Note that if no absences are provided, unsupervised learning will be used anyway.

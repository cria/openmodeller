Subversion

Instructions and repository location

To retrieve the openModeller library sources do:

`svn co https://openmodeller.svn.sourceforge.net/svnroot/openmodeller/trunk/openmodeller`

To retrieve the openModeller Desktop application sources do:

`svn co https://openmodeller.svn.sourceforge.net/svnroot/openmodeller/trunk/openModellerDesktop`

To retrieve the terraView Plugin application sources do:

`svn co https://openmodeller.svn.sourceforge.net/svnroot/openmodeller/trunk/terraViewPlugin`

For activity monitoring and source code statistics, visit our ohloh.net project page.
